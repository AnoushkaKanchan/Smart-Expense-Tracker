Smart Expense Tracker

A web-based expense tracker that helps users manage income, expenses, budgets, and savings on a monthly basis.

Features
- Add and manage income & expenses
- Monthly expense summary
- Budget planning and tracking
- Savings goals and allocation
- LocalStorage-based data persistence
- Clean and responsive UI using Bootstrap

Tech Stack
- HTML5
- CSS3
- Bootstrap
- Vanilla JavaScript
- Local Storage
  
What I Learned
- DOM manipulation and event handling
- State management using JavaScript
- Data persistence using LocalStorage
- Writing modular and readable code

Future Improvements
- Charts & visual analytics
- Export to CSV/PDF
- User authentication
